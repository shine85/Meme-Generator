# 🤪 Simple Meme Generator (纯前端表情包生成器)

一个基于 HTML5 Canvas 的简易表情包生成工具。无需后端，纯浏览器端运行。

## ✨ 功能特点

- 🖼 **实时预览**：上传图片后立即显示效果
- 📝 **文字编辑**：支持顶部和底部文字输入
- 🎨 **样式调整**：经典的“白字黑边”表情包字体，支持调整大小
- 💾 **一键下载**：生成 PNG 格式图片保存到本地
